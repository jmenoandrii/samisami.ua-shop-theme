# SAMISAMI PrestaShop Theme

Custom modern PrestaShop theme developed for **SAMISAMI furniture e-commerce store**.  
The theme focuses on performance, UX, and clean architecture for scalable online stores.

## Features

- Custom product page layout
- Optimized color & variant selectors
- Modular SCSS architecture
- Responsive design (mobile-first)
- Performance-oriented template overrides
- Custom UI components for furniture e-commerce
- Compatible with PrestaShop 8.x / 9.x

## Requirements

- PrestaShop **8.x or 9.x**
- Node.js **>= 20**
- NPM **>= 8**
